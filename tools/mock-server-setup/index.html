Local Mock Server Setup

Start a local Node.js mock API so Postman can receive real HTTP status codes like 404 and 500.

1) Requirements

  • Install Node.js 18+.
  • Open terminal in the project root.

Download script URL:

https://akh826.github.io/tools/mock-server-setup/mock-server.js

Local file link: tools/mock-server-setup/mock-server.js

2) Start Server

Default port is 8787.

node tools/mock-server-setup/mock-server.js

Custom port example:

set PORT=9000 && node tools/mock-server-setup/mock-server.js

3) Test in Postman

GET static file, force HTTP 404:

GET http://localhost:8787/api/errors/not-found.json?__mockStatus=404

POST echo request body, force HTTP 201:

POST http://localhost:8787/api/v1/users?__mockStatus=201

4) Route Rules

  • /api/*.json with GET/HEAD reads files from tools/api-tester/api/.
  • POST/PUT/PATCH/DELETE return JSON echo payload.
  • Use __mockStatus query parameter to override HTTP status code.
