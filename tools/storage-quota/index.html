Site Storage & Cache Inspector

Inspect everything this browser exposes for the current website: localStorage, sessionStorage, cookies, Cache Storage, IndexedDB, service workers, and overall storage quota.

This web page can only read storage for its own origin. To inspect another website, use the browser extension below on that site’s tab. HTTP disk cache cannot be listed from JavaScript.

Inspect other websites

Install the companion extension, open any site, then click the toolbar icon → Inspect active tab (includes HttpOnly cookies).

Download extension (ZIP) Install.bat only (repo clone)

One click after extract: unzip → double-click Install.bat (Windows) or Install.command (Mac). Chrome/Edge opens with the extension loaded. Windows may show SmartScreen — choose “More info” → “Run anyway”.

Manual install (Load unpacked)
  1. Open chrome://extensions → enable Developer mode.
  2. Load unpacked → select the unzipped storage-inspector-extension folder.
