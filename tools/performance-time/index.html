Performance & Time Benchmarks

View page-load stage timings from Navigation Timing API and estimate whether the client clock appears offset from server time.

Clock-drift estimation uses rapid fetch requests with a same-origin Date-header check first, then public UTC time fallbacks if needed. Results are indicative, not a strict replacement for full NTP synchronization.
