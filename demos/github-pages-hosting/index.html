Tips

                    
    
                        
  • Custom domain: add a CNAME file or set a domain under Pages settings.
    • 
                        
  • Relative links: on project sites, use paths like ./about.html or a <base href> tag.
    • 
                        
  • 404 page: add 404.html at the publish root for a custom not-found page.
